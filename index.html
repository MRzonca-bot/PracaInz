Interaktywny system wspomagający decyzję pacjenta w zakresie leczenie i konieczności konsultacji lekarskich.

Niniejszy system został stworzony do celów naukowych i powinien być postrzegany jako takowy element. Autorem niniejszego systemu jest Mirosław Rzońca. Cały system został zrealizowany jako element zaliczeniowy w kierunku tytułu inżynieryjskiego. W systemie nie zostały utworzone bazy danych użytkowników mające na celu gromadzenie informacji. Niniejszy system w podstawowej formie wyklucza jakąkolwiek możliwość manipulacji oraz gromadzenia danych od jego użytkowników. W celu sprawdzenia funkcjonalności systemu naciśnij "Przejdź dalej"
